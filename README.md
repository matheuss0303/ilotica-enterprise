# 👓 IL Ótica Enterprise

Sistema completo de gestão para óticas desenvolvido com React, Node.js e SQLite.

## 📋 Sobre o Projeto

O **IL Ótica Enterprise** foi desenvolvido para otimizar a gestão de óticas, centralizando processos de atendimento, exames, vendas, financeiro, estoque e controle de usuários em uma única plataforma.

O sistema foi projetado para utilização em computadores e tablets, oferecendo uma experiência moderna, intuitiva e responsiva.

---

## 🚀 Funcionalidades

### 🔐 Autenticação e Segurança

* Login por e-mail
* Criação de usuários por convite
* Confirmação de e-mail
* Recuperação de senha por código
* Senhas criptografadas com Bcrypt
* Controle de permissões por perfil

### 👥 Clientes

* Cadastro completo de clientes
* Foto do cliente
* CPF e dados pessoais
* Histórico completo
* Integração com WhatsApp

### 👓 Exames e Receitas

* Cadastro de receitas oftalmológicas
* Controle de O.S.
* Altura da lente
* Tipo de lente
* Médico responsável
* Status de produção
* Histórico de exames

### 💰 Vendas

* Registro de vendas
* Controle de descontos
* Valor pago
* Valor restante
* Controle de pagamento
* Comprovante em PDF

### 📦 Produtos e Estoque

* Cadastro de produtos
* Controle de estoque
* Estoque mínimo
* Atualização automática após vendas

### 📅 Agenda

* Cadastro de agendamentos
* Controle de status
* Organização de atendimentos

### 📊 Financeiro

* Faturamento total
* Ticket médio
* Relatórios financeiros
* Exportação em PDF
* Filtros por período

### 📝 Auditoria

* Registro de ações dos usuários
* Controle de clientes cadastrados
* Controle de exames cadastrados
* Controle de vendas realizadas
